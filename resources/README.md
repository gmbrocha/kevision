# Resources

`resources/` is the intended home for durable product resources.

- `revision_sets/` will eventually hold source drawing sets once path
  compatibility work is finished.
- `sample_outputs/` is reserved for curated, intentionally tracked examples.

For now, `revision_sets/` still lives at the repo root so existing CloudHammer
scripts and tests keep working.
