# Documentation Layout

Status: documentation map as of 2026-05-02.

Canonical application docs live directly under `docs/`:

- `ARCHITECTURE.md`
- `CLIENT_WORKFLOW.md`
- `CURRENT_STATE.md`
- `DATA_FLOW.md`
- `DECISIONS.md`
- `DEPLOYMENT.md`
- `MODULES.md`
- `RUNBOOK.md`
- `SECURITY_POLICY.md`

Reference artifacts live under `docs/references/`. This includes walkthrough
exports, spreadsheets, CSV templates, RFP/reference files, and source/reference
copies of superseded policy docs.

`docs/SECURITY_PRIVACY_POLICY.md` remains only as a compatibility stub for
older notes. The canonical policy summary is `docs/SECURITY_POLICY.md`.

Raw meeting notes live under `docs/meetings/`.

Historical notes live under `docs/history/`.

Visual/product anchor artifacts remain under `docs/anchors/`.

Superseded documentation snapshots live under:

`docs/archive/docs_archive_2026_05_02/`

Do not use root `archive/` for documentation history. Root `archive/` is for
old scripts, experiments, outputs, and implementation artifacts.
