# Documentation Archive - 2026-05-02

This archive was created during the documentation restructure for the
CloudHammer_v2 eval pivot and broader ScopeLedger application clarity.

Old docs are preserved here, not deleted. New canonical docs now live at:

- root-level application docs
- `docs/`
- `CloudHammer_v2/docs/`

The root `archive/` folder is intentionally not used for this documentation
archive. Root `archive/` remains reserved for old scripts, experiments,
outputs, and implementation artifacts. Documentation history belongs under
`docs/archive/` so active docs and archived docs stay together.

## Archive Layout

- `root_docs/`: root-level Markdown files copied before migration/rewrite
- `docs_folder/`: migrated or summarized files from `docs/`
- `cloudhammer_docs/`: CloudHammer-related docs and selected generated
  Markdown summaries that informed the pivot
- `other_migrated_docs/`: reserved for later documentation-only migrations
